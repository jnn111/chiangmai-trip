# 清迈4天3晚旅游攻略

基于宁曼森萨因酒店（Hotel Sensai Nimman）的详细行程规划

## 🌟 特色

- ✅ 基于宁曼路核心位置规划，所有行程都从酒店出发
- ✅ 详细交通信息（车程时间、费用、交通方式）
- ✅ 每天5家精选咖啡店（小红书热门）
- ✅ 小红书必去景点全覆盖
- ✅ 可在线编辑，多人协作
- ✅ 交互式地图，一键查看路线

## 📅 行程概览

| 日期 | 主题 | 亮点 |
|-----|------|------|
| Day 1 | 抵达+古城+夜市 | 塔佩门、契迪龙寺、周日夜市 |
| Day 2 | 粘粘瀑布+网红打卡 | 粘粘瀑布、天使瀑布、大象咖啡 |
| Day 3 | 艺术村+大象+素贴山 | Baan Kang Wat、大象保护营、双龙寺日落 |
| Day 4 | 购物+离别 | 瓦洛洛市场、买咖啡豆、返程 |

**Hotel Sensai Nimman（宁曼森萨因酒店）**
- 📍 地址：110 Sirimangkalajarn Road, Suthep, 清迈

## 🚀 部署到Vercel

### 方法一：通过Vercel CLI部署

1. **安装Vercel CLI**
```bash
npm i -g vercel
```

2. **登录Vercel**
```bash
vercel login
```

3. **进入项目目录并部署**
```bash
cd chiangmai-trip-vercel
vercel --prod
```

### 方法二：通过GitHub部署（推荐）

1. **创建GitHub仓库**
   - 登录GitHub
   - 创建新仓库（如 `chiangmai-trip`）
   - 上传本文件夹内所有文件

2. **连接Vercel**
   - 登录 [vercel.com](https://vercel.com)
   - 点击 "New Project"
   - 导入GitHub仓库
   - 点击 "Deploy"

3. **完成！**
   - Vercel会自动生成网址（如 `chiangmai-trip.vercel.app`）
   - 分享给朋友即可访问

### 方法三：直接拖拽部署

1. 登录 [vercel.com](https://vercel.com)
2. 点击 "New Project" → "Import Git Repository"
3. 选择 "Upload" 选项
4. 拖拽本文件夹内所有文件
5. 点击 "Deploy"

## ✏️ 使用说明

### 编辑功能

1. 点击右上角 **"✏️ 编辑"** 按钮
2. 点击任意文字即可修改
3. 修改完成后点击 **"✓ 完成"** 保存
4. 刷新页面后修改仍然保留（保存在浏览器本地）

### 地图功能

1. 点击 **"🗺️ 查看地图"** 展开地图
2. 点击 **"查看路线"** 查看当天路线
3. 点击景点卡片，地图自动定位

### 预算计算器

- 点击预算数字可直接修改
- 自动计算总花费（泰铢和人民币）
